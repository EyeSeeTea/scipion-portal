  • Click on the alignment reliability box.

    This protocol determines the consistency between a 3D reconstruction from its projections attending to their alignment precision.


    Input

    It needs a reference map (to align noise images) and a set of aligned particles.


    Parameters

    • Symmetry: Symmetry as in xmipp symmetry page


    Output

    Quality parameter which determines the percentage of particles that aligns better than noise

    A chart with information about the P value (comparison between the particle and noise alignment precision) for each particle. A value higher that 1 means that the particle aligns with more precision than noise.

  • Click on the xmipp - validate overfitting box.

    This protocol might tell you whether your set of particles can reconstruct a valid map or not.


    Input

    It needs a reference map (to align noise images) and a set of aligned particles.


    Parameters

    • Resize...(Optional): If obtaining the best possible reconstruction is not your goal, you can resize your input particles and volume to reduce running time of the protocol.
    • Symmetry: Symmetry as in xmipp symmetry page


    Output

    A chart with two series related to the resolution of the reconstructed maps; one from aligned particles and another from aligned noise. Comparing them will help you to see if your aligned particles may reconstruct a valid map or not.
