  • Import your volume file to our server by clicking on the Import Volumes box. Once uploaded, press "Execute"!

  • Import your particles files to our server on the Import particles box. For this particular workflow the particles need to be aligned.   • Downsample the particles to reduce the processing time of the later protocols.

  • Click on the validate non-tilt box.

    Parameters can be easily customized. Two level of usages are supported (Normal and Advanced), which should be chosen according to your experience. Press "Execute"!


  • Click on the xmipp - validate overfitting box.

    This protocol will tell you whether your set of particles can reconstruct a valid map or not.


    Input

    It needs a reference map (to align noise images) and a set of aligned particles.


    Parameters

    • Symmetry: Symmetry as in xmipp symmetry page
    • Number of particles (Advanced expert level): Number of subsets and related number of particles in each subset. e.g.: 50 100 200 500 1000 2000. Note that the biggest set should be less than half of the total input particles.
    • Number of times the randomization is performed: the number of times that reconstruction will be performed for each subset.


    Output

    A chart with two series related to the resolution of the reconstructed maps; one with aligned particles and another with aligned noise. Comparing them will help you to see if your aligned particles may reconstruct a valid map or not.
